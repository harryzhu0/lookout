# Lookout

Lookout is a lightweight, real‑time chat service designed for fast development, easy deployment, and maximum chaos. It provides a simple WebSocket‑based backend paired with a clean web interface, making it ideal for small teams, prototypes, or anyone who wants a chat service without the lag and feel of enterprise email platforms that rhyme with… Outlook.

## How to Run

This is for the few other developers collaborating on this project with me that currently don't know what the duck is happening.

You need to open the terminal (Ctrl + \`) and then type in the terminal, `./chat`. This will execute the program "chat" which is a shell script that runs the server. To stop running the serve, press Ctrl + C, which is a keyboard interrupt that cancels the running of the program.

A further note: To access the website, click on the "ports" tab and click on the port with the number 3000. It will redirect you. Don't click on 8080, because that's the server websocket port and it is not a website clicking it will do nothing.
